	- DevOps ::::
	
		DevOps is a Software Development strategy/process, it helps to build the collaboration between the teams like Development Team and Operations Teams which helps to achieve Continuous Development, Continuous Integration, Continuous Testing, Continuous Delivery, Continuous Deployment and Continuous Monitoring in more automated fashion.
		
	- How to Implement DevOps? 
		
		- Teams involved in the process of SDLC
			
			DevOps/MLOPs Team :
			
				- Infra-Structure Management Team 
				- Application Deployment Team 
				- Testing Team 
				- Release Management Team 
				- Production Support / monitoring Team 
				- IT Security 
				- Data Scientist/Analyst
				- ML Engineers
				
		- Stages of DevOps :
		
			Continuous Development, Continuous Integration, Continuous Testing, Continuous Delivery, Continuous Deployment and Continuous Monitoring.
